# クイックスタート

## 1. インストール

```bash
cd DataCatalog-v01
bun install
```

## 2. サーバー起動

```bash
bun run dev
```

サーバーが http://localhost:3000 で起動します。

## 3. 使い方

### テーブル作成
1. 「+ 新規テーブル作成」ボタンをクリック
2. 必要な情報を入力（システム名、テーブル物理名、テーブル論理名、作成者は必須）
3. 「保存」をクリック

### テーブル詳細表示
- テーブル一覧からテーブルをクリックすると詳細画面が表示されます
- テーブル情報とカラム定義が表形式で表示されます

### カラム追加
1. テーブル詳細画面で「+ カラム追加」ボタンをクリック
2. カラム情報を入力（物理名、論理名、データ型は必須）
3. PK/FK/NULL許可のチェックボックスを設定
4. 「追加」をクリック

### 検索
- 検索バーにキーワードを入力すると、テーブル名、カラム名、概要から部分一致検索
- リアルタイムで検索結果が更新されます

### CSV ダウンロード
- テーブル詳細画面で「CSV ダウンロード」ボタンをクリック
- テーブル情報とカラム定義がCSV形式でダウンロードされます

### テーブル削除
- テーブル詳細画面で「削除」ボタンをクリック
- 確認ダイアログで「OK」を選択すると、テーブルと関連カラムが削除されます

## 4. Docker で起動

```bash
# ビルドと起動
docker-compose up -d

# ログ確認
docker-compose logs -f

# 停止
docker-compose down
```

## 5. 制限事項

- 1テーブルあたり最大500カラムまで登録可能
- データベースは SQLite を使用（catalog.db ファイル）
- デフォルトユーザー: admin (編集者権限)

## トラブルシューティング

### ポート3000が使用中の場合
```bash
# 環境変数でポート変更
PORT=3001 bun run dev
```

### データベースをリセットしたい場合
```bash
rm catalog.db
bun run dev  # 再起動で新しいDBが作成されます
```
